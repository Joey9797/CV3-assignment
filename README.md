## 실행 방법

### 사전 요구사항
- Node.js 18+ (권장)

### 1. 저장소 clone
```bash
git clone https://github.com/Joey9797/CV3-assignment
cd CV3-assignment
```

### 2. 서버 실행 (터미널 1)
```bash
cd server
cp .env.example .env
npm install
npm run dev
```

### 3. 클라이언트 실행 (터미널 2)
```bash
cd client
cp .env.example .env
npm install
npm run dev
```

### 4. 브라우저 접속
→ http://localhost:5173
